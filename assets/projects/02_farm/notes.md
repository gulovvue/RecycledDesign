# Проект 02 — Модульная вертикальная фитоферма (для городской квартиры)

Source: figma file "Eco-подход.fig" — 15 frames.
Author: Данилов Михаил, 1-МГ-50, СПбГУПТД.

## Slides (extracted)

01. Cover — "МОДУЛЬНАЯ ВЕРТИКАЛЬНАЯ ФЕРМА — для городской квартиры"
02. Выбор объекта — Объект редизайна: модульная вертикальная фитоферма, мобильная напольная конструкция для пользователей 65–80 лет. Уход за растениями — часть повседневной жизни пожилых людей; крупные горшки тяжело переставлять, что создаёт нагрузку на спину и риск падения. Существующие решения не учитывают возрастные особенности. Задача — фитоферма, которая позволит безопасно перемещать всю конструкцию на колёсах, ставить и снимать горшки без наклонов.
03. Эко-подход — переработанный пластик rPP/rPET, минимизация материалов, разборная конструкция без клея, замена отдельных элементов, сортировка деталей по мономатериалам.
04. Аналоги — компактные подоконные системы / вертикальные фермы / гидропонные установки / мобильные конструкции.
05. IKEA KRYDDA/VÄXER — компактная настольная LED + гидропоника. Минусы: маленький объём, только для столешницы.
06. GreenStalk — вертикальная башня модульных ярусов, большая ёмкость. Минусы: стационарная, тяжёлая, не зонирует пространство.
07. AeroGarden — гидропоника с автополивом. Минусы: дорогие расходники, сложная электроника, трудный ремонт и утилизация.
08. Образ пользователя — Людмила, 68 лет. Живёт одна, ухаживает за растениями. Тяжело поднимать горшки, неудобно наклоняться. Потребности: перемещать без подъёма, меньше наклоняться, безопасность, самостоятельность.
09. Концепция проекта — Горшки крепятся в индивидуальные чехлы на горизонтальных пазах панели без инструментов. Принципы: перемещение без подъёма, удобный хват, устойчивость, мягкий ход колёс, домашний внешний вид.
10. Устойчивость конструкции — Кашпо 2–4 кг, рама до 40 кг. Низкий центр тяжести, база 100×40 см, равномерное распределение, 4 поворотных колеса.
