# Project 03 — Трансформируемая ремонтная станция для мелкой электроники

**Подход:** Бытовая переработка (Domestic recycling)
**Автор:** Данилов Михаил, 1-МГ-50, СПбГУПТД

## Концепция
Жёсткая упаковка «Золотого Яблока» трансформируется в компактную ремонтную станцию для мелкой электроники без использования дополнительных материалов, кроме элементов самой коробки.

- основание коробки — центральная рабочая поверхность
- крышка — вертикальная задняя панель
- боковые клапаны — ограничители и боковые секции
- полосы из той же коробки — перегородки отсеков

## Целевая аудитория
Молодые пользователи 18–24 лет: студенты, начинающие специалисты. DIY-культура, ремонт смартфонов, наушников, приставок и мелкой электроники в домашних условиях.

**Персона:** Дмитрий, 21 год, студент. Иногда сам разбирает устройства — заменить деталь, почистить разъём или аккумулятор. Проблема: отсутствие удобной рабочей поверхности — детали и винты теряются на столе, инструменты лежат хаотично.

**Потребности:**
- выделенная рабочая площадь для разборки
- отдельные отсеки для винтов, деталей, мелких компонентов
- быстрый доступ к инструментам
- компактное решение, которое можно закрыть и убрать
- использование доступного материала без покупки специальной станции

## Сценарии использования
- **Собрано** — коробка хранится как обычная упаковка
- **В работе** — устройство в центре, винты и детали по отсекам
- **На паузе** — мелкие элементы остаются внутри, не смешиваясь на столе

## Аналоги (slides 5–7)
- iFixit Pro Tech Toolkit — портативный кейс инструментов + съёмный лоток
- Solderbot Portable Soldering Station — нижняя часть рабочее поле, верхняя инструменты
- Portable Technician Case (Tactix) — модульные съёмные контейнеры

## Принцип трансформации
Сборка поэтапная, на основе исходной геометрии: коробка раскрывается → крышка поднимается как задняя панель → боковые элементы формируют ограничители → вставные полосы создают перегородки → центр — рабочая площадь. Фиксация через пазы, загибы, вставки. Раскладываемая, не одноразовая.
